🎮 Como Jogar

Os controles foram configurados para garantir uma experiência confortável e intuitiva, permitindo jogar utilizando ambas as mãos.

Ação	Teclas
Mover para a esquerda	A ou ←
Mover para a direita	D ou →
Pular	W, ↑ ou Espaço

Observação: o personagem só pode pular quando estiver apoiado em uma plataforma sólida.

🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias web modernas:

HTML5
Utilização do elemento <canvas> para renderização gráfica em tempo real.
CSS3
Uso de gradientes lineares (linear-gradient), sombras e fontes externas via Google Fonts.
JavaScript (ES6)
Implementação da lógica do jogo, movimentação do personagem, detecção de colisão, captura de eventos de teclado (keydown e keyup) e loop de renderização otimizado com requestAnimationFrame.
